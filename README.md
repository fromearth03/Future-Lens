# FutureLens — Financial & Economic Trend Prediction System

FutureLens is a prediction system designed to forecast financial and economic trends across multiple domains. The application provides a unified platform to analyze historical data and generate predictions using statistical and mathematical models.

This project focuses on building a practical desktop application capable of forecasting trends such as stock market behavior, cryptocurrency volatility, interest rate changes, and energy consumption patterns.

---

## Features

- Stock Market Trend Prediction  
- Cryptocurrency Volatility Prediction  
- Interest Rate Prediction  
- Population Growth Prediction  
- Commodity Price Prediction  
- Energy Consumption Prediction  
- Disease Spread Prediction  
- Multi-domain prediction system in a single application  
- Graphical User Interface for easy interaction  

---

## Technologies Used

- C++  
- WinForms (.NET)  
- CSV Data Processing  
- Statistical Prediction Models  

---

## Prediction Models Implemented

- Linear Regression  
- Moving Average (SMA / EMA)  
- Markov Chains  
- Monte Carlo Simulation  
- Logistic Growth Models  
- Seasonal Linear Regression  

---

## Data Sources

The application uses historical and real-time data from:

- Alpha Vantage  
- Yahoo Finance  
- CoinMarketCap  
- Pakistan Bureau of Statistics  
- International Energy Agency  

---

## System Architecture

FutureLens is structured into three main components:

### Data Input Module

- Reads historical data from CSV files  
- Fetches data from APIs  
- Preprocesses and validates input data  

### Prediction Engine

- Implements prediction algorithms  
- Processes historical datasets  
- Generates trend predictions  

### User Interface

- Built using WinForms  
- Interactive prediction interface  
- Displays prediction results and insights  

---

## Download & Run

This repository includes the executable version of the application.

### Steps to Run

1. Download files from the **release** folder  
2. Ensure all `.dll` files remain in the same directory as `.exe`  
3. Run `FutureLens.exe`

No additional installation required.
